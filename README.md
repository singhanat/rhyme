# rhyme 

รวมคำคล้องจองไว้เล่นตลก

## ทรงหัวล้าน

ทุ่งหมาหลง  
ดงช้างข้าม  
ง่ามเทโพ  
ชะโดตีแปลง  
แร้งกระพือปีก  
ฉีกขวานฟาด  
ราชคลึงเครา  

## สักยันต์

แขนขวาสักทรงองค์นารายณ์  
แขนซ้ายสักชาติราชสีห์  
เท้าขวาหมึกสักพยัคฆ์คี  
เท้าซ้ายสักหมีมีกำลัง  
พระอุระสักพระโมคลา  
ภควันปิดตาสักเบื้องหลัง  
ที่สีข้างลงอักขระนะจังงัง  
หน้าผากฝังเหล็กกล้าเพชรตาแมว
